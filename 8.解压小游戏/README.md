# 🎮 解压小游戏

一个专为放松心情设计的网页小游戏集合，包含多种解压元素和互动体验。

## ✨ 游戏特色

### 🫧 泡泡纸模拟器
- 点击泡泡听到解压声音
- 自动爆破模式
- 泡泡会自动重置
- 真实的泡泡爆破音效

### 🏖️ 沙子画
- 用鼠标在沙子上画画
- 自定义沙子颜色
- 沙子雨效果
- 支持触摸设备

### 🌊 液体流
- 点击添加彩色液体滴
- 液体自然流动效果
- 自定义液体颜色
- 液体透明度渐变

### ✨ 粒子爆炸
- 点击产生粒子爆炸效果
- 烟花模式
- 可调节粒子数量
- 多种颜色粒子

### 🎵 解压音效
- 8种不同的解压音效
- 可调节音量
- 实时音效反馈
- 支持白噪音

## 🚀 快速开始

1. 直接在浏览器中打开 `index.html` 文件
2. 选择想要体验的游戏模式
3. 享受解压时光！

## 🎯 游戏玩法

### 泡泡纸
- **点击泡泡**: 爆破单个泡泡
- **重置泡泡**: 重新生成所有泡泡
- **自动爆破**: 自动随机爆破泡泡

### 沙子画
- **鼠标拖拽**: 在画布上画画
- **清空画布**: 清除所有沙子
- **沙子雨**: 随机掉落沙子
- **颜色选择**: 改变沙子颜色

### 液体流
- **点击画布**: 添加液体滴
- **添加液体**: 批量添加液体
- **清空液体**: 清除所有液体
- **颜色选择**: 改变液体颜色

### 粒子爆炸
- **点击画布**: 创建爆炸效果
- **烟花模式**: 自动创建多个爆炸
- **清空粒子**: 清除所有粒子
- **粒子数量**: 调节爆炸粒子数量

### 解压音效
- **点击按钮**: 播放对应音效
- **音量控制**: 调节音效音量
- **多种音效**: 泡泡声、雨声、海浪等
